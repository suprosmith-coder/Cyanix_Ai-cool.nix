Sure! Here's a Python function to calculate Fibonacci sequence with memoization for efficiency:
fibonacci.py 
def fibonacci(n, memo={}):
    """Calculate nth Fibonacci number with memoization."""
    if n in memo:
        return memo[n]
    if n <= 2:
        return 1
    memo[n] = fibonacci(n-1, memo) + fibonacci(n-2, memo)
    return memo[n]

# Example usage
for i in range(1, 11):
    print(f"Fibonacci({i}) = {fibonacci(i)}")
